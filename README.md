# IRAAP Frontend

Frontend for the IRAAP (Intelligent Repository for Academic Projects) system. A modern web application built with Next.js, TypeScript, and Tailwind CSS that allows students to submit projects, supervisors to review submissions, and users to search the project archive.

## Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **State Management**: TanStack Query (React Query)
- **HTTP Client**: Axios
- **Authentication**: JWT (context-based)
- **Icons**: Lucide React
- **Package Manager**: pnpm

## Prerequisites

- Node.js v18 or higher
- pnpm (`npm install -g pnpm`)
- Backend API running (see [IRAAP Backend](https://github.com/Megabyte-webdev/IRAAP-BACKEND))
